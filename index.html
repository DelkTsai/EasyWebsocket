Easy WebSocket

like WebSocket but no server setup and available in any browser

EasyWebSocket aims to make realtimes webapps in every browser without the trouble to setup servers.

How to Use It

Include the following in your webpage and it just works. 

    <script src="http://jeromeetienne.github.com/EasyWebsocket/web/easyWebSocket-min.js">
    </script>	
    <script>
        var socket = new EasyWebSocket("ws://example.com/resource");
        socket.onopen	= function(){
            socket.send("hello world!")
        }
        socket.onmessage= function(event){
            alert("received "+ event.data)
        }
    </script>

No server setup, no cross-origin issue to care about... nothing!

You are done, see it live.

The API is copied on the standard API, thus compatible and easy to learn.
