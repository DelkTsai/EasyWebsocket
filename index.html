EasyWebsocket

like Websocket but no server setup and available in any browser

Websocket Anywhere aims to make realtimes webapps in every browser without the trouble to setup Websocket servers.

How to Use It

Just include the following in your webpage. The API is copied on the standard API see it live 

    <script src="http://jeromeetienne.github.com/EasyWebsocket/web/easyWebSocket-min.js">
    </script>	
    <script>
        var socket = new EasyWebSocket("ws://example.com/resource");
        socket.onopen	= function(){
            socket.send("hello world!")
        }
        socket.onmessage= function(event){
            alert("received "+ event.data)
        }
    </script>
